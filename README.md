# Hierarchical Category x Subcategory Registry

Static GitHub Pages bundle for the interactive Raiz hierarchical category x
subcategory registry.

## Files

- `index.html` — interactive registry viewer.
- `UNIFIED_CAT_SUBCAT_REGISTRY.pdf` — printable PDF snapshot.

## Publish

Create a public GitHub repo, push this folder as the repo root, then enable
GitHub Pages from the `main` branch root.
